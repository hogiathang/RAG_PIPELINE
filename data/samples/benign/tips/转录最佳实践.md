# 转录最佳实践

---

## 零、技术选型

### FunASR Paraformer

阿里开源，中文识别最优，支持字符级时间戳。

---

## 一、环境准备

```bash
# 1. 安装依赖
pip install funasr modelscope

# 2. 下载模型（首次使用）
cd /path/to/videocut-skills/安装/scripts
python test_funasr_local.py --download
```

---

## 二、使用方法

```bash
cd /path/to/videocut-skills/剪口播/scripts

# 转录视频
python transcribe_local.py video.mp4

# 指定输出路径
python transcribe_local.py video.mp4 --output=result.json
```

---

## 三、输出格式

### JSON 结构

```json
{
  "input_file": "video.mp4",
  "mode": "local",
  "full_text": "完整转录文本...",
  "duration_s": 217.97,
  "chars": [
    { "char": "大", "start": 0.88, "end": 1.12 },
    { "char": "家", "start": 1.12, "end": 1.36 },
    { "char": "好", "start": 1.36, "end": 1.60 }
  ]
}
```

### 字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| input_file | string | 输入文件路径 |
| mode | string | 使用的模式 (local) |
| full_text | string | 完整转录文本 |
| duration_s | number | 音频时长（秒） |
| chars | array | 字符级时间戳 |
| chars[].char | string | 单个字符 |
| chars[].start | number | 开始时间（秒） |
| chars[].end | number | 结束时间（秒） |

---

## 四、工作原理

```
1. 30s 分段提取音频
    ↓
2. 每段调用 Python FunASR
    ↓
3. 合并结果 + 时间偏移校正
    ↓
4. 返回统一格式
```

### 为什么用 30s 分段

| 方案 | 问题 |
|------|------|
| FunASR 全视频 | 长视频时间戳漂移（~10s/3分钟）→ 剪辑不准 |
| **FunASR 30s分段** | ✅ 无漂移 + 精确时间戳 |

### 模型说明

| 模型 | 用途 |
|------|------|
| `paraformer-zh` | 中文语音识别主模型 |
| `fsmn-vad` | 检测哪里有人说话 |
| `ct-punc` | 自动添加标点符号 |

## 五、音频预处理

转录前自动执行：

```bash
ffmpeg -i video.mp4 \
  -vn \                    # 不要视频
  -acodec pcm_s16le \      # 16-bit PCM
  -ar 16000 \              # 16kHz 采样率
  -ac 1 \                  # 单声道
  output.wav
```

### 参数说明

| 参数 | 值 | 原因 |
|------|-----|------|
| 采样率 | 16000 Hz | 模型训练采样率 |
| 声道 | 单声道 | 语音识别不需要立体声 |
| 格式 | WAV | 无损，兼容性好 |
| 位深 | 16-bit | 足够，文件更小 |

## 七、常见问题

### Q1: 报错 "funasr not found"

**解决**：

```bash
pip install funasr modelscope
```

### Q2: 时间戳和文字对不上

**原因**：标点符号没有时间戳

**解决**：转录模块已自动跳过标点

### Q3: 识别准确率低

**可能原因**：
- 背景噪音太大
- 说话人口音重
- 音频采样率不对

**解决**：
- 预处理降噪
- 确保音质清晰

### Q4: pip 和 python 版本不一致

**解决**：确保 pip 和 python 指向同一个版本

```bash
# 检查版本
python --version
pip --version

# 如果不一致，在 ~/.zshrc 添加 alias
alias python=python3.11
alias pip=pip3.11
```

## 八、Token对齐规则

### Token规则

| 元素 | 时间戳 |
|------|--------|
| 中文字符 | 每字1个 |
| 英文单词 | 整词1个（`agent` 算1个） |
| 标点/空格 | 无时间戳（已过滤） |

### 访问时间戳

```python
import json

with open('transcript.json', 'r') as f:
    result = json.load(f)

for item in result['chars']:
    print(f"{item['char']}: {item['start']}s - {item['end']}s")
```
